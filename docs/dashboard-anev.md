# Dashboard ANEV Polres

Halaman **/anev/polres** menampilkan rekap aktivitas, kepatuhan, serta performa konten lintas platform. Per Juni 2025, normalisasi payload ANEV untuk **TikTok per satfung/divisi** membaca lebih banyak variasi field dan menampilkan metrik tambahan (views, likes, comments, shares, engagement rate) agar operator bisa menilai kualitas interaksi, bukan sekadar volume posting.

## Normalisasi data

## UI ringkas
- Dashboard ANEV Polres kini menyediakan tombol **Export Excel** yang mengekspor data sesuai filter aktif (`time_range`, `start_date`, `end_date`, `role`, `scope`, `regional_id`, `client_id`). File berisi ringkasan KPI, posting per platform, compliance per pelaksana, user per satfung/divisi, TikTok per satfung/divisi, likes Instagram per satfung/divisi, direktori user, serta engagement Instagram/TikTok per user dalam satu workbook. Implementasi memakai route khusus `/api/dashboard/anev/export` dengan generator `exceljs`; data dipecah otomatis ke beberapa sheet berdasarkan `section` sehingga laporan lebih mudah diaudit per domain metrik.
- UI filter juga menyediakan tombol cepat **Export Harian / Export Mingguan / Export Bulanan**. Saat tombol dipilih, sistem otomatis menerapkan preset periode terkait, mengambil data terbaru ANEV, lalu mengunduh Excel agar operator bisa mengekspor tanpa langkah manual tambahan.
- Seksi **TikTok per Satfung/Divisi** kini menampilkan badge Engagement Rate (ER) ketika tersedia, kartu posting/engagement utama, serta chip kecil untuk views, likes, comments, dan shares jika dikirim backend.
